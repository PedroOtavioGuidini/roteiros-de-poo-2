a)O que quebrou?
O método Validar da ValidadorSenha ganhou um novo parâmetro obrigatório, então o Console que chamava senha.Validar(senhaDigitada) parou de compilar.

b)Breaking Change
É qualquer alteração em uma biblioteca que quebra o código de quem já a utilizava. Exemplos: mudar nome de método, adicionar parâmetro obrigatório, remover um método.
